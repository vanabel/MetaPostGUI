# 例子来源与许可

## tlhiv.org

- **页面**：[Metapost : examples](http://www.tlhiv.org/MetaPost/examples/examples.html)
- **用途**：编译回归与解析覆盖率语料（`examples/tlhiv/`）
- **说明**：经典 MetaPost 教程式片段；figure 正文抽取自页面 `beginfig` 块，不含外层 `beginfig`/`endfig`。

## thruston/metapost-examples

- **仓库**：[thruston/metapost-examples](https://github.com/thruston/metapost-examples)
- **许可**：GNU General Public License v3.0
- **用途**：
  - `examples/thruston/` —  vendored 工具宏源文件（只读参考）
  - `config/plugins/` — 适配为 MetaPostGUI 插件的副本（`*.plugin.json` + `*.mp`）
- **未纳入 GUI 例子库**：`pww-1.tex`、`pww-2.tex`、`cosmos.tex`、`excursions.tex`（需 `luamplib` / LuaLaTeX）

## MetaPostGUI 精选

- `examples/curated/` — 使用 `scaled u` 约定、面向画布同步的精选例子（MIT，与项目相同）
